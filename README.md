# Tools Store Front-End (Shop)

Tienda online optimizada para la venta de herramientas profesionales. Enfocada en la conversión, velocidad de carga y posicionamiento SEO.

## 🚀 Tecnologías
- **React 19 & Vite**: Motor principal.
- **TypeScript**: Estructura de datos sólida.
- **Tailwind CSS**: Estilos responsivos.
- **React Helmet Async**: SEO dinámico por producto.
- **Cloudinary SDK**: Optimización de imágenes en tiempo real.
- **Zustand**: Carrito de compras persistente.

## 🛠️ Variable de Entorno (Vercel)
Indispensable para conectar con el inventario real:

```env
VITE_API_URL=https://tu-api-prod.vercel.app/api/v1
```

## ✨ Características Destacadas
- **Filtros Avanzados**: Búsqueda por precio, marca y categoría.
- **SEO Ready**: Metadatos dinámicos para Google y Redes Sociales.
- **Checkout Inteligente**: Soporte para cupones de descuento y múltiples medios de pago.
- **Optimización**: Imágenes en formatos ligeros (WebP) automáticos.

## 📦 Instalación
1. `npm install`
2. `npm run dev`
